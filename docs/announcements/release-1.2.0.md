# 1.2.0：平台中心和语雀导入优化

这个版本主要围绕“平台越来越多以后如何更好维护”和“语雀导入稳定性”做升级。

## 更新内容

- 新增平台中心入口，按平台展示导出、导入和教程能力。
- 新增任务中心，方便查看最近任务和错误报告。
- 优化语雀 Markdown 导入，增强图片路径识别。
- 语雀导入支持接口超时、上传超时、失败重试和只重试失败项。
- 生成语雀导入计划时会输出 `00-语雀导入计划.json`，方便定位缺失图片和远程图片。

## 建议

语雀批量导入前，先“生成计划”，再“单篇导入测试”，最后批量导入。这样可以先发现图片缺失、远程图片不可访问或登录过期等问题。
